Améliorez l'interface d'un site mobile avec des animations CSS

Le quatrième projet d'OpenClassroom a été très bénéfique dans mon apprentissage du developpement web. En effet, il s'agit de développer un site en mobile first en utilisant Sass et des animations css

Sass est un language très proche du CSS avec de nombreux avantages. Il permet de créer des variables, des règles encapsulées, des mixins, des fonctions et même d'utiliser des opérations mathématiques

Les animations CSS permettent de créer des transitions entre deux états de mise en forme. Une animation est décrite par deux choses : des propriétés propres à l'animation d'une part et un ensemble d'étapes (keyframes) qui indiquent l'état initial, final et éventuellement des états intermédiaires d'autre part. Trois avantages permettent de distinguer les animations CSS des techniques d'animations utilisant JavaScript : On peut aisément obtenir des animations simples sans avoir à connaître JavaScript. Les animations s'exécuteront correctement même lorsque le système est soumis à une charge modérée. Il est possible que des animations JavaScript s'exécutent lentement si elles sont mal décrites. Dans le cadre des animations CSS, le moteur de rendu peut utiliser certaines techniques (comme le frame-skipping) afin que le résultat obtenu soit aussi fluide que possible. En laissant le contrôle de l'animation au navigateur, celui-ci peut optimiser les performances et l'efficacité du système, par exemple en réduisant la fréquence de mise à jour des animations qui sont exécutées dans des onglets qui ne sont pas visibles à l'écran.

Ce projet m'a permit d'avoir une nouvelle approche du code, notamment en CSS grace à la manipulation de SASS et des animations CSS ainsi que de toutes leurs fonctionnalités.

Implémentez le front-end d'une application bancaire avec React

Dans le cadre de ce projet, j'ai développé le front-end d'une application bancaire en utilisant React. Mon objectif était de créer une interface utilisateur dynamique et réactive pour offrir une expérience utilisateur optimale.

Grâce à ce projet, j'ai pu mettre en pratique mes compétences en développement front-end en utilisant React et Redux. J'ai créé une architecture solide pour gérer l'état global de l'application et faciliter la communication entre les différents composants.

J'ai intégré le contenu de l'application en utilisant React Router, ce qui m'a permis de gérer la navigation de manière fluide et de créer des routes spécifiques pour les différentes fonctionnalités de l'application.

J'ai également développé la gestion de l'authentification des utilisateurs, en permettant la connexion et la déconnexion, ainsi que l'accès au profil utilisateur après une authentification réussie. J'ai également implémenté la fonctionnalité permettant à l'utilisateur de modifier son pseudo.

En résumé, ce projet m'a permis de développer mes compétences en développement web avec React, Redux et React Router, tout en créant une interface utilisateur réactive et conviviale pour une application bancaire complète. J'ai appris à gérer l'état global de l'application et à gérer l'authentification des utilisateurs.
